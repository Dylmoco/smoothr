# storefronts

Client-facing storefront code and widgets.

This package contains the core storefront modules located in `./core`.

## Core modules

- abandoned-cart
- affiliates
- analytics
- currency
- dashboard
- discounts
- orders
- returns
- reviews
- subscriptions

### Usage

```javascript
import * as core from './core/index.js';

core.discounts; // example access
```

# features

Storefront features for Smoothr.

## Modules

- abandoned-cart
- affiliates
- analytics
- currency
- dashboard
- discounts
- cart
- orders
- returns
- reviews
- subscriptions
- auth

### Usage

Import modules directly from their paths:

```javascript
import { orders } from './orders/index.js';

orders.renderOrders(); // access the orders module
```
